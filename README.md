# 📊 Student Spending Analysis Dashboard (Power BI)

## 📌 Project Overview

This project analyzes student spending behavior across different store settings (Urban, Suburban, Rural) in the United States using Power BI, focusing on category-wise purchase patterns and store performance.

---

## 🎯 Objectives

* Analyze total purchase (TAP) across store location and setting
* Identify spending trends across age groups
* Compare category-wise spending behavior
* Understand impact of store setting on purchases

---

## 📊 Dashboard Features

### 🔹 TAP Overview

* Total purchase by store location and setting
* Color-coded:

  * 🔴 < 35,000 | 🟡 35,000–60,000 | 🔵 ≥ 60,000

### 🔹 Outdoor Spend

* Matrix view of outdoor sports spending by Age and Store Setting

### 🔹 Purchase Funnel

* Total purchase by Store Setting
* Shown as Percentage of First

### 🔹 Suburban Spend

* Store location-wise distribution for Suburban setting

### 🔹 Games vs Sports

* Scatter plot comparing Video Games and Outdoor Sports spending

### 🔹 Indoor vs Video

* Comparison of Indoor Games and Video Games across age groups

---

## 🔐 Row-Level Security

* Role-based access to restrict data by Store Setting

---

## ☁️ Deployment

* Published on Power BI Service with scheduled refresh
